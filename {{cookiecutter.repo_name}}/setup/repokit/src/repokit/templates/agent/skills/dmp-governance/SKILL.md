# dmp-governance

## Workflow
1. Register datasets:
   - repokit-dmp dataset
2. Normalize/update DMP:
   - repokit-dmp update
3. Edit or validate manually when needed:
   - repokit-dmp editor

## Verify
- dmp.json structure is valid and up to date.
- Dataset metadata references match actual local/remote resources.
